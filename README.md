# html-authetication-poc
This project is a simple authentication flow created using only HTML.

It contains 5 basic web pages connected using anchor tags for navigation.

## Files Included

1. login.html  
2. register.html  
3. forgot-password.html  
4. reset-password.html  
5. dashboard.html  

## Page Flow

- Login page redirects to Dashboard
- Register page redirects to Login
- Forgot Password page redirects to Reset Password
- Reset Password page redirects back to Login
- Dashboard has a Logout link that redirects to Login

## Technology Used

- HTML only
- No CSS
- No JavaScript
